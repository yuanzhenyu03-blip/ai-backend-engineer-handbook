# Final Capstone

## Goal

Build the final portfolio project that combines Python, FastAPI, Playwright, infrastructure, and AI backend concepts into one interview-ready system.

## Learning Objectives

- Design a complete backend system from requirements to deployment.
- Combine API services, workers, data storage, automation, and AI features.
- Practice production documentation and system design explanation.
- Prepare a project story for overseas startup interviews.

## Planned Features

- FastAPI backend
- Authentication
- PostgreSQL persistence
- Redis-backed jobs or caching
- Playwright automation workflow
- AI feature integration
- Docker deployment
- Test suite and documentation
