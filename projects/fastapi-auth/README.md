# FastAPI Auth

## Goal

Build a production-style authentication service with FastAPI, JWT, dependency injection, and testable architecture.

## Learning Objectives

- Design authentication APIs with clear request and response models.
- Practice FastAPI dependency injection.
- Implement JWT-based login and protected routes.
- Separate routing, services, schemas, and persistence concerns.

## Planned Features

- User registration
- User login
- JWT access token generation
- Protected profile endpoint
- Password hashing
- Unit and integration tests
