# Playwright Login

## Goal

Build a robust login automation project that handles browser context, storage state, selectors, waiting, and error recovery.

## Learning Objectives

- Use Playwright Locator API correctly.
- Manage browser contexts and storage state.
- Avoid brittle sleeps.
- Design maintainable page objects.

## Planned Features

- Login flow automation
- Storage state reuse
- Explicit waiting
- Page object structure
- Error screenshots
- Tests or dry-run checks
