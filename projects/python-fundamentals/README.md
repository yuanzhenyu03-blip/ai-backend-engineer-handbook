# Python Fundamentals

## Goal

Build a production-style Python fundamentals project that turns core language design topics into tested, readable code examples.

## Learning Objectives

- Practice Python 3.12+ syntax with type hints.
- Understand references, mutability, functions, decorators, context managers, and async code through implementation.
- Write small modules that are easy to test and review.
- Connect Python design knowledge to backend engineering interviews.

## Planned Features

- Object model examples
- Mutable vs immutable examples
- Iterator and generator examples
- Decorator examples
- Context manager examples
- Async examples
- Unit tests for every topic
