# Playwright Scraper

## Goal

Build a production-style scraping project that uses Playwright safely, handles dynamic pages, and returns structured data.

## Learning Objectives

- Use browser, context, page, and locator correctly.
- Extract structured data from dynamic websites.
- Handle retries, timeouts, and expected failures.
- Keep scraping logic modular and testable.

## Planned Features

- Configurable target pages
- Locator-based extraction
- Structured output models
- Retry and timeout handling
- Logging and diagnostics
- Tests for parser logic
