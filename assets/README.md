# Assets

Assets support lessons, projects, architecture explanations, and interview preparation.

Do not store generated images or diagrams in random folders.

```text
assets/
├── images/
├── diagrams/
├── architecture/
└── memory-model/
```

## Folder Responsibilities

- `images/`: screenshots and general visual assets
- `diagrams/`: reusable diagrams for lessons and projects
- `architecture/`: system architecture sketches
- `memory-model/`: Python memory model and object model visuals

No images are required yet. These folders are reserved for future visual material.
